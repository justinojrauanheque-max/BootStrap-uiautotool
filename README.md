# Bootstrap-uiautotool

Bootstrap loader for Auto Clicker - Form Helper web assets. This repository contains the core bootstrap script and HTML files for the Auto Clicker - Form Helper extension.

## Files

- **bootstrap.js** - Main bootstrap loader that dynamically loads all CSS and JavaScript resources from Supabase
- **index.html** - Landing page with documentation, test section, and privacy policy
- **options.html** - Configuration interface for managing automation settings

## Features

- Dynamic asset loading from Supabase storage
- Support for CodeMirror editor with syntax highlighting
- Internationalization support (i18n)
- Responsive design with mobile support
- Fallback mechanisms for failed resource loads

## Installation

1. Clone this repository
2. Deploy files to your web server or CDN
3. Update Supabase base URL in bootstrap.js if needed
